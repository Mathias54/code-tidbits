# ES6 Shorthand to Define Methods in Objects

Awesome JS tip that helps you type less! 👍

Use ES6’s concise method syntax to define methods in your objects.

```javascript
const Rain = {
  // bringUmbrella: function() {}
  
  bringUmbrella() {
    return '☔️'
  }
}
```
