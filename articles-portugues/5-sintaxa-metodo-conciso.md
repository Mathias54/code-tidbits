# Forma Rápido de Definir Métodos em Objetos ES6 

Essa dica incrível de JS ajuda você a escrever menos! 👍 

Use a sintaxe "Concise Methods" para definir métodos em seus objetos.

```javascript
const Chuva = {
  // levarGuardaChuva: function() {}
  
  levarGuardaChuva() {
    return '☔️'
  }
```
